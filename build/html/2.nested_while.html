2. More About While Loops¶

Nested While Loops

A nested loop means a loop inside another loop. Think of it like a box within a box. But in our case, we have an outer while loop and in that we have an inner while loop. The inner loop runs completely for each iteration of the outer loop. This is useful for situations where you need to perform a repetitive task that itself contains repetitive tasks.

Why use nested while loops?

Nested while loops are particularly useful for:

  • Iterating over multi-dimensional data structures, like matrices.

  • Performing repetitive tasks that involve multiple layers of iteration.

outer loop while condition:
  inner loop while condition:
      # statement inside inner while loop
  # statement inside outer while loop

Worked Example 1:¶

Let’s create a simple program that uses a nested while loop to print the following pattern:

1 1

1 2

2 1

2 2

i = 1
while i < 3:
    j = 1
    while j < 3:
        print(i, j)
        j += 1
    i += 1

This code initializes the variable i to 1 and enters a while loop that continues as long as i is less than 3. Within this loop, another variable j is set to 1, and an inner while loop runs as long as j is less than 3. Each iteration of the inner loop prints the current values of i and j, then increments j by 1. After the inner loop finishes, i is incremented by 1. When i reaches 3, the outer loop stops, ending the program.

            print("Invalid password, try again")
